# spotipi

This is a project intended as a birthday present for my brother.

It is a chassis for a Raspberry Pi Zero 2 W that is built to accomodate a [sound card](https://www.microcenter.com/product/631852/iqaudio-dac-sound-card-for-raspberry-pi-4b-3b-3b).

It is also a Docker container that contains [spotifyd](https://github.com/Spotifyd/spotifyd),
and is intended to be run in [Fedora IoT](https://fedoraproject.org/iot/).
